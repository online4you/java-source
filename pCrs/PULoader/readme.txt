1.- ENGLISH

#### System requirements ####

- 512 MB of RAM 

- Java virtual machine installed and running.

#### How to start ####

To start the application you need to execute the file 'run' located at /bin directory.

- Windows o.s. --> run.bat
- Linux o.s. --> run.sh

To schedule the launch of application you need to execute the file 'runCommandMode' located at /bin directory.

- Windows o.s. --> runCommandMode.bat
- Linux o.s. --> runCommandMode.sh

Note that you must edit the configuration file (confCommandMode/PUCommandModeConf.properties) to define your local properties.


2.- ESPAÑOL

#### Requisitos del sistema ####

- 512 MB de RAM 

- Máquina virtual de java instalada y en marcha.

#### Cómo empezar ####

Para lanzar la aplicación debe ejecutar el fichero 'run' que encontrará en el directorio /bin

- S.O. Windows --> run.bat
- S.O. Linux --> run.sh

Para lanzar la aplicación de manera planificada y sin entorno gráfico, debe ejecutar el fichero 'runCommandMode'

- S.O. Windows --> runCommandMode.bat
- S.O. Linux --> runCommandMode.sh

Tenga en cuenta que deberá preparar adecuadamente la configuración (confCommandMode/PUCommandModeConf.properties) para su correcta ejecución.
